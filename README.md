# 🎬 Live Sports - بث مباشر

موقع احترافي لبث المباريات المباشرة والأفلام والمسلسلات

## ✨ المميزات

- 🔴 **بث مباشر للمباريات** - شاهد المباريات المباشرة الآن
- 🎥 **أفلام** - مجموعة كبيرة من الأفلام
- 📺 **مسلسلات** - أحدث المسلسلات
- 📅 **جدول زمني** - معرفة مواعيد المباريات القادمة
- 📱 **تصميم متجاوب** - يعمل على جميع الأجهزة
- ⚡ **سريع وبدون تقطيع** - أداء عالي

## 🛠️ التكنولوجيا المستخدمة

- **HTML5** - هيكل الموقع
- **CSS3** - تصميم وتنسيق جميل
- **JavaScript** - التفاعلية والوظائف
- **Responsive Design** - يعمل على جميع الأحجام

## 📂 هيكل الملفات

```
Ahmad/
├── index.html      # الصفحة الرئيسية
├── style.css       # التنسيقات والأنماط
├── script.js       # الوظائف التفاعلية
└── README.md       # هذا الملف
```

## 🚀 كيفية الاستخدام

1. **افتح الموقع:**
   - افتح ملف `index.html` في متصفحك

2. **تصفح المحتوى:**
   - استخدم شريط التنقل العلوي للتنقل بين الأقسام
   - اضغط على أي زر "شاهد الآن" للمشاهدة

3. **تابع المباريات:**
   - اطلع على قسم المباريات المباشرة
   - تحقق من الجدول الزمني للمباريات القادمة

يمكنك تخصيص الموقع بسهولة:

2. **إضافة محتوى:** عدّل `index.html` وأضف المزيد من البطاقات
3. **إضافة وظائف:** عدّل `script.js` وأضف وظائف جديدة
