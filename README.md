# متجر أحمد الجديد

## نظرة عامة

واجهة إدارية احترافية (Admin Dashboard) متخصصة في إدارة تطبيقات الهاتف والمشتركين، خاصة نظام iOS.

---

## 🎯 المميزات الرئيسية

### 📱 إدارة تطبيقات iOS
- إدارة شاملة لتطبيقات iOS
- التحكم الكامل بإصدارات التطبيق

### 👥 إدارة المشتركين والاشتراكات
- إضافة وإزالة المشتركين
- متابعة حالة الاشتراكات
- تحديث بيانات الاشتراك

### 📲 إدارة أجهزة المستخدمين
- ربط الأجهزة بحسابات المستخدمين
- إضافة أو إزالة الأجهزة المرتبطة
- تتبع أجهزة كل مستخدم

### 🔐 إدارة أكواد التفعيل والشهادات
- توليد وإدارة أكواد التفعيل
- إدارة الشهادات الخاصة بالتطبيق
- التحكم بصلاحيات التفعيل

### 📊 متابعة البيانات والإحصائيات
- تتبع بيانات المستخدمين
- مراقبة حالة الاشتراكات
- تقارير شاملة عن الأداء

---

## ⚙️ طبيعة النظام

هذا النظام جزء من:

- **منصات إدارة تطبيقات اشتراكية** (Subscription-based Platforms)
- **أنظمة التوزيع الداخلية للتطبيقات** (Enterprise / In-house Apps)
- **أنظمة التحكم بالمستخدمين والخدمات** (User & Service Management Systems)
