# 📱 دليل تشغيل النظام على الجوال

## المتطلبات
- النظام يعمل على `http://localhost:3000`
- جوال متصل بنفس شبكة الويفاي

## الخطوات

### 1️⃣ الحصول على IP الجهاز
**على Windows/Linux:**
```bash
ipconfig /all  # Windows
# أو
ifconfig      # Linux/Mac
```

**ابحث عن:** IPv4 Address (مثال: `192.168.1.100`)

### 2️⃣ على الجوال
- افتح المتصفح
- اذهب إلى: `http://<YOUR-IP>:3000`
- مثال: `http://192.168.1.100:3000`

### 3️⃣ تسجيل الدخول
```
البريد: admin@ahmad.local
كلمة المرور: Admin123!
```

### 4️⃣ إذا لم يعمل
أدخل عنوان IP يدويًا في صفحة الدخول:
- في قسم **"عنوان السيرفر (للجوال)"**
- اضغط **"حفظ العنوان"**
- جرب الدخول مجددًا

## استكشاف الأخطاء

### ❌ لا يمكن الوصول إلى الخادم
```bash
# تأكد من أن الخادم يستمع على جميع الواجهات
node backend/server.js

# يجب أن ترى:
# 🚀 Ahmad Admin Dashboard
# 📱 الجوال: http://<your-ip>:3000
```

### ❌ CORS Error
- تم إصلاح هذا تلقائيًا في `backend/server.js`
- امسح ذاكرة المتصفح وأعد التحميل

### ❌ الجوال يشهد توقفًا
- تأكد من اتصال Wifi
- تحقق من IP الصحيح
- أعد تشغيل الخادم

## الاختبار السريع

```bash
# 1. ابدأ الخادم
npm start

# 2. من جهاز آخر على الشبكة
curl http://192.168.1.100:3000/api/health
# يجب أن ترى: {"status":"ok","timestamp":"..."}
```

## 🎯 للإنتاج
إذا كنت تريد تشغيل النظام من الإنترنت:
- استخدم Ngrok أو خدمة مشابهة
- أو انشر على Heroku/Railway/Render
